# Support

## Where to get help

### Slack

The [#dagster-alerts](https://kippnj.slack.com/archives/C04A0KC1YSW) channel is
the primary place for pipeline failure notifications and data operations
discussion. Post questions about Dagster runs, dbt errors, or data issues here.

### Asana

Pull requests automatically create tasks in the
[Teamster Asana Project](https://app.asana.com/0/1205971774138578/1205971926225838).
Use this for tracking code review and deployment status.

### GitHub Issues

For bugs, feature requests, or documentation improvements, open an issue at
[github.com/TEAMSchools/teamster/issues](https://github.com/TEAMSchools/teamster/issues).
